# アクセシビリティ

## 概要

Darts Lab のアクセシビリティ対応状況。WCAG 2.1 AA 準拠を目標とする。

## 監査ツール

- **axe-core**: Playwright E2E テストに統合 (`e2e/accessibility.spec.ts`)
- **Storybook a11y アドオン**: コンポーネントレベルの a11y チェック
- **Chrome DevTools Lighthouse**: ページレベルの a11y スコア

## テスト実行方法

```bash
# E2E でアクセシビリティテストを実行
npx playwright test e2e/accessibility.spec.ts

# Storybook の a11y パネルで確認
npm run storybook
```

## 対応状況

### 実施済み

- 全インタラクティブ要素に `aria-label` を付与
- ランドマーク要素の適切な使用（`nav`, `main`, `footer`）
- フォーム要素のラベル関連付け
- テーマ切替ボタンのアクセシブル表示
- モバイルドロワーの `role="navigation"` 付与
- 画像の alt テキスト
- 色のみに依存しない情報伝達（アイコン + テキスト）

### 既知の課題

- カラーコントラスト: ダークテーマの一部でコントラスト比が不十分な箇所あり
- フォーカスインジケーター: MUI デフォルトに依存（カスタマイズ余地あり）
- スキップナビゲーション: 未実装
